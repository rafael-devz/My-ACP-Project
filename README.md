# Grill Bite - FOOD ORDERING SYSTEM

Hello People!
This Project was created for our Advanced Computer Programming (ACP) subject using Python and CustomTkinter.

About this project:
Grill Bite is a simple food ordering system beginner type where users can:
- Register an account
- Login to the system
- Browse food menu
- Add items to the cart and place orders
- Choose a payment method like foodpanda
- And view order history

While doing this project, it gaves me a help to make more familiarize and practice:
- Python Programming
- GUI Development with CustomTkinter
- Store data using file handling
- Make the system user friendly by making input validations

I hope someday my project will give ideas to others and make them inspire.
Thank you for visiting my project!
